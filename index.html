เชื่อมต่อ Firebase
ตั้งค่าครั้งเดียว จากนั้นซิงค์ทุกอุปกรณ์อัตโนมัติ
1
ไปที่ console.firebase.google.com → สร้าง Project ใหม่ (ฟรี)
2
เลือก Build → Firestore Database → Create → เลือก Test mode
3
ไปที่ Project Settings (⚙️) → Your apps → เพิ่ม Web app (</>) → Copy config ที่ได้มา
4
วาง config ด้านล่างแล้วกด เชื่อมต่อ
